# linux-file-permission-management-lab
This project demonstrates my ability to inspect, interpret, and manage file permissions in a Linux environment using command-line tools. As part of a simulated cybersecurity scenario, I audited file and directory permissions, identified unauthorized access, and updated permissions to align with organizational access control requirements.
